Ocean Data: About



The Ocean Data app pulls live data from metocean monitoring stations around south west England, UK. Data is provided by Channel Coast Oberservatory

Forecast chart data is pulled from the NOAA Wavewatch III server and processed in python using matplotlib.

Interactive charts were created using the opensource JS library www.chartjs.org
